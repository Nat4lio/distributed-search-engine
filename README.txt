Manual to execute the program

1 - mvn clean install
2 - launch url_queue_run.java
3 - launch ServingWebApplication.java
4 - launch at least one BarrelImpl.java (each time that execute creates a new barrel)
5 - launch the downloader.java

with all theses steps done the program its running at his full power at localhost:8080
http://localhost:8080
